Rozwiązania zadań dotyczących JavaScript

1. Kiedy powstał JavaScript?

JavaScript powstał w 1995 roku.

2. Dlaczego jest to język skryptowy?

JavaScript jest językiem skryptowym, ponieważ jest interpretowany, a nie kompilowany. Oznacza to, że jego kod jest wykonywany bezpośrednio przez przeglądarkę internetową, bez potrzeby kompilacji do kodu maszynowego.

3. Gdzie jest wykonywany JS (klient lub serwer)?

JavaScript może być wykonywany zarówno po stronie klienta (w przeglądarce), jak i po stronie serwera (przy użyciu środowisk takich jak Node.js).

4. Dlaczego JS to bezpieczne narzędzie?

JavaScript jest bezpieczny, ponieważ ma ograniczone uprawnienia dostępu do zasobów komputera, na którym jest uruchamiany. Wszystkie działania w ramach tego języka są kontrolowane przez przeglądarkę.

5. Czy jest to język obiektowy, jeśli tak to dlaczego?

Tak, JavaScript jest językiem obiektowym, ponieważ pozwala na definiowanie obiektów, które mają swoje właściwości i metody. Wspiera model DOM (Document Object Model), który umożliwia interakcję z elementami na stronie jako obiektami.

6. Wymień zalety JavaScript:
  • Zaleta 1: Prosty i łatwy do nauczenia, co sprawia, że szybko można zacząć tworzyć aplikacje.
  • Zaleta 2: Powszechnie stosowany w Internecie, dostępne są miliony przykładów i zasobów.
  • Zaleta 3: Odpowiedni do tworzenia dynamicznych i interaktywnych aplikacji, które reagują w czasie rzeczywistym.
7. Framework to:

Framework to zbiór gotowych narzędzi, bibliotek i struktur programistycznych, które ułatwiają tworzenie aplikacji poprzez dostarczenie gotowych rozwiązań dla typowych problemów.

8. Do aplikacji mobilnych można użyć Framework:

Tak, do tworzenia aplikacji mobilnych można używać frameworków takich jak React Native, który umożliwia tworzenie aplikacji na różne systemy operacyjne.

9. Aplikacje responsywne to:

Aplikacje, które dostosowują swój wygląd i układ do różnych rozmiarów ekranów, dzięki czemu dobrze wyglądają i działają zarówno na komputerach, jak i na urządzeniach mobilnych.

10. Fragment kodu:

‹input type="button" name="przycisk" value="Nowa Strona" onclick="WinOpen(' ')"›

Ten fragment kodu HTML tworzy przycisk z napisem "Nowa Strona". Po jego kliknięciu wywoływana jest funkcja JavaScript o nazwie WinOpen, która prawdopodobnie otwiera nowe okno przeglądarki (w zależności od definicji funkcji).

11. Fragment kodu:

window.open("obraz.html", "okienko", "toolbar=no,directories=no,menubar=no,height=280,width=160,top=200,left=200");

Ten kod otwiera nowe okno przeglądarki z plikiem HTML o nazwie "obraz.html". Nowe okno nosi nazwę "okienko", a jego wygląd jest skonfigurowany następująco:

  • toolbar=no: Ukrywa pasek narzędzi.
  • directories=no: Ukrywa przyciski katalogów.
  • menubar=no: Ukrywa pasek menu.
  • height=280: Ustawia wysokość okna na 280 pikseli.
  • width=160: Ustawia szerokość okna na 160 pikseli.
  • top=200: Ustawia pozycję okna 200 pikseli od górnej krawędzi ekranu.
  • left=200: Ustawia pozycję okna 200 pikseli od lewej krawędzi ekranu.
12. Instrukcja window.close()

Instrukcja window.close() zamyka bieżące okno przeglądarki, w którym została wywołana. Aby działała, musi być wywołana w oknie otwartym za pomocą JavaScript (np. window.open()).
